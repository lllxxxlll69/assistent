Что нужно сделать:
1. Установить зависимости:
   pip install -r requirements.txt

2. Положить русскую модель Vosk в папку model рядом с main.py.
   Пример структуры:
   main.py
   app/
   model/
     am/
     conf/
     graph/
     ivector/

3. Запустить Ollama и убедиться, что модель доступна.
   По умолчанию используется qwen3-vl:4b

4. Запустить приложение:
   python main.py

Если папка модели называется не model, можно задать переменную окружения:
Windows cmd:
   set VOSK_MODEL_PATH=имя_папки_или_полный_путь

PowerShell:
   $env:VOSK_MODEL_PATH="имя_папки_или_полный_путь"
